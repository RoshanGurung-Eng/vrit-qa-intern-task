# Vrit Technologies QA Intern Task - Signup Automation

## 📋 Overview

Automates the signup flow on https://authorized-partner.vercel.app/ using Selenium WebDriver with Python programming language.

## ▶️ How to Run

```bash
# 1. Clone repository
git clone https://github.com/RoshanGurung-Eng/vrit-qa-intern-task.git
cd vrit-qa-intern-task

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the script
python scripts/signup_automation.py
```
